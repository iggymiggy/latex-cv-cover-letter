# CV and Cover Letter Template

## Setup

Edit `personal_info.tex` with your information, then compile.

## Files

- `personal_info.tex` - Your personal information
- `cv/CV-template.tex` - CV document
- `cover_letter/cover_letter_CV_style.tex` - Cover letter

## Compiling

**LaTeX Workshop extension:**
- Install "LaTeX Workshop" in your editor
- Save `.tex` file to auto-compile, or use build command
- View PDF with preview command

**Command line:**
```bash
cd cv && pdflatex CV-template.tex
cd ../cover_letter && pdflatex cover_letter_CV_style.tex
```
